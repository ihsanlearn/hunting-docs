# Reconnaissance

#### Output dari Reconnaissance

1. Host / Subdomain yang alive
2. Endpoint (URL / API) yang relevan + method + parameter
3. Javascript Files
4. Potensi titik masuk (file path parameter, redirect parameter, URL fetch parameter, upload endpoints)
5. Fingerprint teknologi dan proteksi (WAF, CDN, server type, rate limit)
6. Kandidat takeover / leak / misconfig
